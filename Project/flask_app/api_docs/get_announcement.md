# Get Announcement

Get announcement made so far

**URL** : `/api/announcement`

**Method** : `GET`

**Auth required** : NO


## Success Response

**Code** : `200 OK`

**Content example**

```json
[
    {
        "time": "12/01/19 10:22",
        "title": "Test Announcement",
        "body": "This is a test announcement"
    },
    {
        "time": "12/01/19 10:36",
        "title": "Test Announcement2",
        "body": "This is a test announcement"
    }
]
```
